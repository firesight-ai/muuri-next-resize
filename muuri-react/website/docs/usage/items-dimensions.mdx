## Dimensions when dragging ‚úã

When the Items are dragged they are inserted into the [dragContainer](../api-reference/muuricomponent#dragcontainer). If their dimensions are relative to the parent (e.g.&nbsp;`width: 70%;`) the drag event could cause some unwanted resizing of the Items. <br/>
For these cases you can use the prop [dragFixed](../api-reference/muuricomponent#dragfixed). If used, the CSS `width`, `height` and `padding-top` properties of the element will be fixed while the Item is dragging.

```jsx
<MuuriComponent dragEnabled dragFixed />
```

### Example üìñ

Simple example showing the width of an Item, assuming that the MuuriComponent element width is `1000px`.

```css
.muuri-item {
  width: 70%;
}
```

|           | No drag | Drag start | Drag end |
| --------- | ------- | ---------- | -------- |
| Css width | 70%     | 700px      | 70%      |
